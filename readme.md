Install our application dependencies
```
$ npm i
```

Spin docker instance (redis)
```
$ npm run docker-compose:up
```

Destroy docker instance
```
$ npm run docker-compose:down
```

Run web application
```
$ npm run serve
```

To run (basic usage):
```
$ npm i
$ npm run docker-compose:up
$ npm run serve

open http://localhost:3000
```
